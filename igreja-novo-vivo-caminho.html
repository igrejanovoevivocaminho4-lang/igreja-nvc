⚙ Como configurar o banco de dados em nuvem

1. Acesse console.firebase.google.com com sua conta Google
2. Clique em Criar projeto → dê um nome (ex: igreja-nvc)
3. No menu lateral, clique em Firestore Database → Criar banco de dados → modo Produção
4. Vá em Configurações do projeto (ícone de engrenagem) → Seus apps → clique em </> (Web)
5. Copie os dados do firebaseConfig e cole nos campos abaixo
6. Em Regras do Firestore, cole as seguintes regras:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /membros/{id} { allow read, write: if true; } match /pendentes/{id} { allow read, write: if true; } } }
